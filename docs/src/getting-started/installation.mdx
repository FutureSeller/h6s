# Installation

```sh
yarn add @veccu/react-calendar
# or
npm install --save @veccu/react-calendar
```

## peerDependencies

```json
"peerDependencies": {
  "date-fns": ">= 2",
  "react": "^16.8.0 || ^17.0.0"
},
"peerDependenciesMeta": {
  "react-dom": {
    "optional": true
  },
  "react-native": {
    "optional": true
  }
}
```
